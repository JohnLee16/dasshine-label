# Dasshine Label 前端界面说明

## 设计概览

Dasshine Label 前端采用**深色科技感主题**，以蓝紫色渐变为主色调，体现专业、高效、智能的产品定位。

### 设计特色

- **品牌标识**：Logo 区域清晰展示 "Dasshine Label" 字样
- **深色主题**：保护视力，适合长时间标注工作
- **科技感**：发光边框、渐变效果、网格背景
- **交互友好**：悬停效果、动画过渡、即时反馈

---

## 页面结构

### 1. 登录页 (`/login`)

- 深色背景 + 网格纹理
- 居中 Logo + 品牌展示
- 渐变边框卡片
- 输入框悬停发光效果

### 2. 工作台 (`/`)

- 欢迎区域（带品牌 Logo）
- 统计数据卡片（今日任务、完成数、准确率、收益）
- 快速开始（项目快捷入口）
- 等级进度展示

### 3. 项目管理 (`/projects`)

- 项目列表表格
- 进度可视化
- 成员数量统计
- 状态管理

### 4. 任务列表 (`/tasks`)

- 状态筛选（待领取/进行中/审核中/已完成）
- 优先级标签
- 进度条显示
- 一键开始标注

### 5. 标注工作区 (`/annotate/:taskId`) ⭐核心

**布局：**
```
┌─────────────────────────────────────────────────────┐
│ 顶部工具栏 (返回 | 项目名称 | 进度 | 计时 | 提交)      │
├──────────────────────────┬──────────────────────────┤
│                          │                          │
│    文本标注区 (8列)       │    标签选择区 (4列)       │
│    - 高亮实体显示         │    - 标签按钮网格         │
│    - 点击选中文本         │    - 当前标注列表         │
│    - 点击删除标注         │    - 添加/删除操作        │
│                          │                          │
│    AI预标注提示           │    快捷操作按钮           │
│                          │                          │
└──────────────────────────┴──────────────────────────┘
```

**交互流程：**
1. 左键选中文本
2. 右侧选择标签
3. 点击"添加标注"
4. 已标注实体显示在文本中
5. 点击实体可删除

## 技术栈

- **框架**: React 18 + TypeScript
- **构建**: Vite 5
- **样式**: Tailwind CSS 3
- **UI库**: Ant Design 5（定制深色主题）
- **图标**: Lucide React
- **状态**: React Query + Zustand
